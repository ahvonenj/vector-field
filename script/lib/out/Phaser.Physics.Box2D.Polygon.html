Class: Polygon

.Box2D. Polygon

new Polygon()

A generic polygon class. Includes functions for decomposing into convex polygons. Use one of the 'set' functions to define the vertices.

decomposeOptimal() → {Array}

Returns an array of the individual convex polygons which make up the given concave polygon. This will return a near-optimal (lowest number of sub-polygons) decomposition and is astoundingly slow for polygons with more than about 8 vertices. You most likely will not want to use this at runtime, but it could be useful as a pre-process for something. Please try the normal 'decompose' function instead.
